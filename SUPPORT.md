# Support

TidyDock is an early open-source project.

## Before Asking For Help

- Check [README.md](README.md).
- Check [docs/README.md](docs/README.md).
- Check [src/TidyDock/README.md](src/TidyDock/README.md).
- Run the portable verification script when reporting package issues:

```powershell
powershell -ExecutionPolicy Bypass -File '.\src\TidyDock\scripts\Test-Portable.ps1' -SkipLaunch
```

## Useful Paths

```text
%APPDATA%\TidyDock\config\settings.json
%APPDATA%\TidyDock\cache\icons

```
